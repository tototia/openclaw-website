# OpenClaw 网站部署指南

## 🎯 最快方案：部署到 Vercel（推荐）

### 步骤1：准备工作
确保你已经安装了：
- Git
- Node.js 18+
- pnpm（或npm）

### 步骤2：将代码推送到 GitHub

```bash
# 初始化git仓库（如果还没有）
git init

# 添加所有文件
git add .

# 提交代码
git commit -m "feat: 创建OpenClaw展示网站"

# 推送到GitHub（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/openclaw-website.git
git branch -M main
git push -u origin main
```

### 步骤3：部署到 Vercel

#### 方式A：通过网页部署（最简单）

1. 访问 [vercel.com](https://vercel.com)
2. 使用GitHub账号登录
3. 点击 "Add New" → "Project"
4. 选择你的 openclaw-website 仓库
5. Vercel 会自动检测 Next.js 项目
6. 点击 "Deploy" 按钮
7. 等待2-3分钟，部署完成！

#### 方式B：通过命令行部署

```bash
# 安装 Vercel CLI
pnpm add -g vercel

# 登录 Vercel
vercel login

# 部署到生产环境
vercel --prod
```

### 步骤4：获取访问链接

部署完成后，Vercel 会提供一个免费的HTTPS链接：
```
https://你的项目名.vercel.app
```

你可以：
1. 将此链接分享给学生
2. 在微信公众号菜单中添加这个链接
3. 自定义域名（需要购买域名）

## 📱 微信公众号集成方案

### 方案1：在公众号菜单添加链接

#### 前提条件：
- 已注册微信公众号（服务号或订阅号）
- 已备案的域名（Vercel提供的是未备案域名）

#### 操作步骤：

1. **登录微信公众平台**
   - 访问 [mp.weixin.qq.com](https://mp.weixin.qq.com)
   - 使用管理员账号登录

2. **配置自定义菜单**
   - 进入"内容与互动" → "自定义菜单"
   - 点击"创建菜单"
   - 添加菜单项：
     - 菜单名称：OpenClaw
     - 跳转网页：粘贴你的 Vercel 链接

3. **发布菜单**
   - 点击"保存并发布"
   - 学生关注公众号后即可点击访问

### 方案2：创建微信公众号内嵌网页（需要备案域名）

#### 操作步骤：

1. **域名备案**
   - 购买域名（如阿里云、腾讯云）
   - 完成ICP备案（需要2-4周）

2. **配置自定义域名**
   - 在 Vercel 项目设置中添加自定义域名
   - 在域名DNS解析中添加CNAME记录

3. **配置公众号网页授权**
   - 进入微信公众平台 → 开发 → 接口权限
   - 配置网页授权域名

4. **创建自定义菜单**
   - 同方案1

### 方案3：转换为微信小程序（最推荐）

#### 优点：
- 无需域名备案
- 体验更好
- 可以利用微信生态

#### 操作步骤：

1. **注册微信小程序**
   - 访问 [mp.weixin.qq.com](https://mp.weixin.qq.com)
   - 点击"立即注册" → 选择"小程序"
   - 填写信息并完成认证（个人主体免费）

2. **创建小程序项目**
   - 下载微信开发者工具
   - 创建新项目，选择"JavaScript"模板
   - 将你的网页内容转换为小程序页面

3. **发布小程序**
   - 在开发者工具中上传代码
   - 在微信公众平台提交审核
   - 审核通过后发布

4. **分享给学生**
   - 小程序码
   - 微信搜索小程序名
   - 在公众号中关联小程序

## 🌐 其他部署方案

### Netlify（免费）

```bash
# 构建
pnpm build

# 安装 Netlify CLI
pnpm add -g netlify-cli

# 登录
netlify login

# 部署
netlify deploy --prod
```

### 腾讯云 CloudBase（国内访问快）

1. 创建腾讯云账号
2. 开通 CloudBase 服务
3. 安装 CloudBase CLI
4. 部署静态网站

### 阿里云 OSS + CDN

1. 购买阿里云OSS存储
2. 构建项目：`pnpm build`
3. 上传 .next 文件夹到OSS
4. 配置CDN加速

## 🎓 教学场景建议

### 如果只是临时分享给学生（1-2周）：
- ✅ 使用 Vercel 免费部署
- ✅ 直接分享链接
- ⏱️ 耗时：5-10分钟

### 如果长期教学使用：
- ✅ 购买域名并部署到 Vercel
- ✅ 在微信公众号菜单中添加链接
- ✅ 考虑转换为微信小程序
- ⏱️ 耗时：1-2天（域名备案需要时间）

### 如果需要国内访问速度：
- ✅ 部署到腾讯云或阿里云
- ✅ 配置CDN加速
- ⏱️ 耗时：1-3小时
