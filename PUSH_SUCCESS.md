## 🚀 下一步：部署到Vercel

现在你可以将网站部署到Vercel了！

### 快速部署步骤（5分钟）：

#### 1. 访问Vercel
打开浏览器，访问：https://vercel.com

#### 2. 登录
使用GitHub账号登录（选择 "Continue with GitHub"）

#### 3. 导入项目
- 点击右上角 "Add New" → "Project"
- 在 "Import Git Repository" 列表中找到 `openclaw-website`
- 点击 "Import" 按钮

#### 4. 配置项目
- Vercel会自动识别Next.js项目
- 确认配置信息（默认即可）
- 点击 "Deploy" 按钮

#### 5. 等待部署
- 部署时间：2-3分钟
- 部署完成时会显示访问链接

#### 6. 获取访问链接
部署成功后，你会得到这样的链接：
```
https://openclaw-website.vercel.app
```
