## 📱 方案1：腾讯云 CloudBase（最推荐）

### 优点
- ✅ 国内访问速度快
- ✅ 有免费额度（每月10GB流量）
- ✅ 腾讯出品，稳定可靠
- ✅ 支持自定义域名
- ✅ 适合教学使用

### 部署步骤（5分钟）

#### 第1步：注册腾讯云账号
1. 访问：https://cloud.tencent.com/
2. 点击"免费注册"
3. 使用手机号注册
4. 完成实名认证（需要身份证，约5分钟）

#### 第2步：开通 CloudBase 服务
1. 访问：https://console.cloud.tencent.com/tcb
2. 点击"新建环境"
3. 填写信息：
   - **环境名称**: `openclaw-website`（自定义）
   - **计费模式**: 选择"按量付费"（有免费额度）
   - **套餐**: 选择"基础版 1"（免费）
   - **环境地域**: 选择"上海"或"广州"（离学生近）
4. 点击"立即开通"

#### 第3步：进入静态网站托管
1. 进入刚创建的环境
2. 左侧菜单找到"静态网站托管"
3. 点击"立即使用"

#### 第4步：上传网站文件

**方法A：使用 CLI 工具（推荐）**

1. 安装 CloudBase CLI：
```bash
npm install -g @cloudbase/cli
```

2. 登录：
```bash
tcb login
```

3. 初始化项目：
```bash
tcb init
```
选择：
- 环境：选择刚创建的环境
- 项目名称：`openclaw-website`
- 项目目录：当前目录

4. 部署：
```bash
tcb hosting deploy out
```

5. 访问网站：
部署成功后会显示访问地址，类似：
```
https://你的环境ID.service.tcloudbase.com
```

**方法B：使用网页上传（简单）**

1. 在 CloudBase 控制台，点击"上传文件"
2. 选择 `out/` 目录下的所有文件
3. 点击"上传"
4. 等待上传完成

#### 第5步：配置域名（可选）

**使用免费域名**：
- CloudBase会自动提供免费域名
- 格式：`https://你的环境ID.service.tcloudbase.com`
- 直接分享给学生即可

**使用自定义域名**（需要购买）：
1. 在 CloudBase 控制台点击"添加域名"
2. 输入你的域名（如 `openclaw.yourdomain.com`）
3. 配置 DNS 解析（按提示操作）
4. 等待DNS生效（10-30分钟）

#### 第6步：分享给学生
将访问地址分享给学生即可：
```
https://你的环境ID.service.tcloudbase.com
```

## 📱 方案2：阿里云 OSS（备选）

### 优点
- ✅ 访问速度最快
- ✅ 非常稳定
- ✅ 支持CDN加速

### 部署步骤

#### 第1步：注册阿里云
1. 访问：https://www.aliyun.com/
2. 注册并完成实名认证

#### 第2步：开通 OSS
1. 进入"对象存储 OSS"
2. 创建Bucket
3. 地域选择"华东1（杭州）"或"华东2（上海）"
4. 访问权限选择"公共读"

#### 第3步：上传文件
1. 进入Bucket
2. 点击"上传文件"
3. 选择 `out/` 目录下的所有文件
4. 上传

#### 第4步：配置静态网站
1. 在Bucket设置中找到"静态网站"
2. 开启静态网站
3. 设置默认首页：`index.html`
4. 获取访问地址

#### 第5步：绑定域名（可选）
1. 购买域名（阿里云域名约50元/年）
2. 配置DNS解析
3. 绑定到OSS

### 费用
- 存储费用：约0.12元/GB/月
- 流量费用：约0.5元/GB
- 月成本：约5-10元

## 📱 方案3：Gitee Pages（完全免费）

### 优点
- ✅ 完全免费
- ✅ 国内访问稳定
- ✅ 操作简单

### 缺点
- ⚠️ 只支持静态网站
- ⚠️ 不支持自定义域名（需要实名认证）

### 部署步骤

#### 第1步：注册 Gitee
1. 访问：https://gitee.com/
2. 注册账号

#### 第2步：创建仓库
1. 点击"+" → "新建仓库"
2. 仓库名：`openclaw-website`
3. 设置为"公开"
4. 初始化仓库

#### 第3步：推送代码
```bash
git remote add gitee https://gitee.com/你的用户名/openclaw-website.git
git push gitee main
```

#### 第4步：启用 Pages
1. 进入仓库
2. 点击"服务" → "Gitee Pages"
3. 启用 Pages
4. 选择分支：`main`
5. 点击"启动"

#### 第5步：获取访问地址
Gitee会提供访问地址：
```
https://你的用户名.gitee.io/openclaw-website
```

### 费用
- ✅ 完全免费
